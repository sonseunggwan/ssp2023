Project 2 Voiced, Unvoiced, and Silence detection on TIMIT dataset

  • attached:

    1. timit_wav.tar: see Projec 1 at LMS1

  • 목적:

    1. Voiced (유성음), Unvoiced (무성음), Silence (묵음) 을 구분하는 분류기를 만든다.
    2. 모든 모음은 유성음, 자음중 /s/, /f/, /th/, /ch/, /k/, /p/, /t/ 은 무성음

  • TIMIT transcripts explanation 모든 wav file 들은 phoneme 단위로 시간정보가 기재되어 있음

    Phonetic symobol list: phoneme classification, see tables 2 and 3

  • Train/Validation/Test

    1. Train/Validation: timit/train의 화자들을 8:2, 혹은 다른 비율로 적당히 나눔 dr_를 dialect region 을 나타냄. 따라서 각 dialect 안의 화자들을 나누는 것이 적합
    2. Test: timit/test

  • 실험

    1. SA (dialect sentences) 로 학습-validation-test
    2. SX (phonetically compact) 로 학습-validation-test

  • 방법

    1. EPD or VAD 를 이용하여 1차 구분: 음성구간 앞뒤 확장 필요
    2. autocorrelation 을 이용하여 voiced/unvoiced rough 구분, a[1]/a[0] <> 0.5, threshold 0.5 변경하면서
    3. ZCR, median filter, heuristic 적용
    4. RNN 사용

  • 제출

    1. Report: 사용한 방법 상세히 설명, 20% test set 에 대한 성능
    2. Code: 실행 가능한 .py or .ipynb, argument는 TIMIT wav 위치.
